其中 Python 安装时需要勾选添加环境变量。

重启。

重启完之后在终端安装 Python 库。Windows 端测试使用的版本如下：

pip install bleak==0.22.3 bless==0.2.6 bleak-winrt==1.2.0 -i https://pypi.tuna.tsinghua.edu.cn/simple
pip install https://github.com/gwangyi/pysetupdi/archive/refs/heads/master.zip

其中：
- bleak-winrt：Windows 下 bleak 访问 WinRT 蓝牙接口需要，客户端和服务端均建议安装
- bless 0.2.6：服务端创建 BLE GATT 服务需要
- pysetupdi 2018.10.22：Windows 下 bless/bleak-winrt 相关依赖需要
- pywin32 311：由 bless 依赖自动安装
